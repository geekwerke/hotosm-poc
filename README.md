POC for HOTOSM FMTM project.

- The [`api`](./api) folder has the code for the Python backend.
- The [`ui`](./ui) folder has the code for the React frontend.

## Running the backend

```
$ cd api/src
$ pdm venu activate
$ uvicorn main:app
```
